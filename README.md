# 👜 PocketBon

> PWA for personal use (개인용 웹앱)

주머니 속에 담긴 나만의 앱 모음 — 설치 없이 브라우저에서 바로 사용하는 개인용 PWA입니다.

---

## ✨ 기능 목록

### 🧮 계산기
- 사칙연산, 퍼센트, 부호 변환

### 🌤️ 날씨
- 도시 이름으로 실시간 날씨 조회
- 기온, 체감온도, 습도, 풍속 표시
- Open-Meteo 무료 API 사용

### ⏰ 알람
- 시간 설정 및 알람 이름 지정
- 켜기/끄기 토글
- 알람 울릴 때 소리 + 화면 팝업
- 브라우저 푸시 알림 지원
- Firebase FCM 연동 (백그라운드 알림)

### ⏱️ 타이머 / 스톱워치
- 카운트다운 타이머
- 스톱워치

### 📝 메모
- 빠른 메모 작성 및 저장

### 💰 가계부
- 수입 / 지출 기록
- 잔액 계산

### 📅 달력 & 일정
- 월별 달력 보기
- 날짜 클릭 → 일정 등록
- 제목, 시작/종료 시간, 색상 선택
- 달력에 색깔 점으로 일정 표시
- 공휴일 자동 표시 (한국천문연구원 API)
- 대체공휴일 포함

---

## 🛠️ 기술 스택

| 항목 | 내용 |
|------|------|
| 프론트엔드 | HTML, CSS, Vanilla JS |
| PWA | Service Worker, Web App Manifest |
| 날씨 API | Open-Meteo (무료) |
| 공휴일 API | 한국천문연구원 특일정보 (공공데이터포털) |
| 푸시 알림 | Firebase Cloud Messaging (FCM) |
| 배포 | Vercel |

---

## 📱 설치 방법 (홈 화면에 추가)

1. 크롬(Android) 또는 사파리(iOS)로 접속
2. 브라우저 메뉴 → **홈 화면에 추가**
3. 아이콘 생성 → 앱처럼 실행 🎉

---

## 🔑 API 키 설정

`index.html` 에서 아래 부분을 찾아 키를 입력하세요:

```js
// 공휴일 API (공공데이터포털)
const HOLIDAY_API_KEY = '여기에_API키_입력';
```

## 🚀 배포

GitHub에 푸시하면 Vercel이 자동으로 재배포합니다.

```
GitHub push → Vercel 자동 빌드 → 배포 완료
```
